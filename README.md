# E-Commerce Full Stack Project (React + Django)

A full-stack e-commerce web application built using React.js frontend and Django REST Framework backend with PostgreSQL database. It includes authentication, product listing, cart system, search, and checkout functionality.

## Tech Stack

Frontend:
- React.js
- Tailwind CSS
- React Router

Backend:
- Django REST Framework
- Python
- PostgreSQL

Other:
- JWT Authentication
- Git & GitHub

  ## Features

- User Registration & Login (JWT Auth)
- Product Listing
- Product Search
- Product Details Page
- Cart Management
- Checkout System
- Category Filtering
- Order API (Backend)
